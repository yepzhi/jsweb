# SIIP NextGEN STEM Diagnostic Test

A high-fidelity, interactive STEM competency assessment tool built for SIIP NextGEN. This diagnostic test evaluates scientific literacy based on international PISA standards.

## Features

- **Premium UI:** Glassmorphism design system with dark mode aesthetics.
- **Categorized Assessment:** 80-point test covering Innovation, Cosmology, Physics, Biology, and more.
- **Interactive Logic:** Smooth transitions, real-time progress tracking, and animated results breakdown.
- **PISA Aligned:** Evaluation framework based on global scientific literacy benchmarks.

## Data Structure

The test questions are managed in `questions.js`, allowing for easy updates and categorization.
